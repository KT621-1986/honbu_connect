# 本部Connect - セットアップガイド

## 必要なもの
- **Node.js** (v18以上)  
  → https://nodejs.org/ja/ からインストール

---

## セットアップ手順

### 1. フォルダをわかりやすい場所に置く
例: `C:\Users\あなたの名前\honbu-connect\`

### 2. ターミナル（コマンドプロンプト）を開く
フォルダを右クリック → 「ターミナルで開く」
または、コマンドプロンプトで以下を入力:
```
cd C:\Users\あなたの名前\honbu-connect
```

### 3. 必要なライブラリをインストール
```
npm install
```
（初回のみ。数分かかります）

### 4. サーバーを起動
```
node server.js
```

### 5. ブラウザでアクセス
```
http://localhost:3000
```

## データについて
- データは `honbu_connect.db` ファイルに保存されます（SQLite）
- サーバーを止めてもデータは保持されます
- バックアップする場合は `.db` ファイルをコピーしてください

---

## サーバーの止め方
ターミナルで `Ctrl + C` を押す

---

## 複数PCで使いたい場合
サーバーを起動しているPCのIPアドレスを確認して、
他のPCからは `http://[IPアドレス]:3000` でアクセスできます。

IPアドレスの確認方法:
```
ipconfig   （Windows）
ifconfig   （Mac/Linux）
```
